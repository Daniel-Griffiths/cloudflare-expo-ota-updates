# CLI Management Tool

The project includes a powerful CLI for managing your OTA updates, with both interactive and direct command modes.

## Usage

**Interactive Mode**

```bash
pnpm run cli
```

**Direct Commands**

```bash
pnpm run cli <command>
```

## Available Commands

| Command        | Description                                  |
| -------------- | -------------------------------------------- |
| `create-app`   | Create a new app with auto-generated API key |
| `list-apps`    | View all created apps                        |
| `list-updates` | Browse updates for a specific app            |
